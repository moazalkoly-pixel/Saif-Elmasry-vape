Saif 2lMasry Vape Store — صفحة روابط
-------------------------------------
محتويات:
- index.html  --> الصفحة الأنيقة جاهزة للرفع
- README.txt  --> تعليمات سريعة لرفع الصفحة على GitHub Pages أو أي استضافة

كيفية رفع الصفحة على GitHub Pages (الخيار المجاني الأكثر سهولة):
1. سجل دخولك إلى GitHub وزر "New repository". سمّه مثلاً: saif-2lmasry-site
2. ارفع ملف index.html (يمكن سحب وإفلاته في صفحة رفع الملفات).
3. من إعدادات المستودع (Settings) > Pages > اختر الفرع 'main' و folder '/' واضغط Save.
4. بعد دقائق ستظهر لك رابط الموقع على github.io مثل
5. استخدم هذا الرابط لإنشاء كود QR (مثال سريع عبر Google Charts API):
   https://chart.googleapis.com/chart?cht=qr&chs=400x400&chl=PASTE_YOUR_SITE_URL_HERE

إذا تحب، أقدر أساعدك بخطوات رفع المستودع أو أجهز ملف ZIP جاهز للتحميل — الملف هنا مرفق.
